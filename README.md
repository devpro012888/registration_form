```markdown
# Free CodeCamp Project - Registration Form

This is a responsive registration form built as part of the Free CodeCamp Responsive Web Design Certification curriculum. The form is designed to collect basic user information and demonstrates key HTML and CSS concepts including form inputs, semantic HTML, and responsive layout.

## 🔧 Features

- User input fields for:
  - First Name
  - Last Name
  - Email
  - Password
- Account type selection (Personal or Business)
- File upload (Profile Picture)
- Age input with validation (13–120)
- Dropdown for referral source
- Bio text area
- Terms and Conditions checkbox
- Submit button styled for responsiveness

## 🛠️ Built With

- **HTML5** for structure
- **CSS3** for styling
- Responsive design principles (viewport width scaling)
- Semantic elements and accessibility best practices

## 💡 Highlights

- Uses `fieldset`, `label`, and `legend` to organize form inputs semantically.
- Implements required fields and input validation with attributes like `required`, `pattern`, `min`, and `max`.
- Uses radio buttons and checkboxes for user selection.
- Submits form data to a test endpoint:  
  `https://register-demo.freecodecamp.org`

## 📁 File Structure
```

└── styles.css       # External CSS file

